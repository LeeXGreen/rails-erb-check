rails-erb-check
====

A command to check syntax as rails's erb.
It provides the rails-erb-check command.

![http://i.imgur.com/QZP4V.png](http://i.imgur.com/QZP4V.png)

Usage
----

    $ gem install rails-erb-check
    $ rails-erb-check app/views/**/*.html.erb
